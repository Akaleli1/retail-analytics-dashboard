# Retail Analytics & Customer Segmentation Dashboard

An end-to-end data project that simulates a real-world retail data pipeline. This tool ingests raw transactional data, performs data cleaning and transformation using **Python (Pandas)** and **SQL**, and visualizes Key Performance Indicators (KPIs) via an interactive **Streamlit** dashboard.

### Key Features
* **Data Engineering:** Automated cleaning and schema standardization of raw CSV data.
* **Data Analysis:** SQL-based querying to extract sales trends and top-performing products.
* **Visualization:** Interactive charts and metrics for business stakeholders.
